## Executive Summary

The UT Computational Nuclear Engineering group requests AWS cloud compute resources to support
an active portfolio of simulation-intensive research programs. Current on-campus infrastructure
imposes hard scheduling constraints that create multi-week queues for large-scale Monte Carlo
transport and thermal-hydraulic simulations. Cloud compute resources will eliminate these
bottlenecks and directly enable competitive grant deliverables.

This proposal requests a **12-month AWS allocation** sufficient to support the research program
described in §1. The requested resources are scoped to cover peak workloads during active
simulation campaigns, with costs estimated from historical job profiles (see §3).

### Program Context

The computational workloads supported by this request include:

- Neutronics simulations for reactor design and safety analysis
- Thermal-hydraulic transient modeling at facility scale
- Coupled multi-physics calculations for advanced fuel performance
- Monte Carlo uncertainty quantification studies requiring thousands of independent runs
